Namco PC System Boards

This page contains specifications of Namco Arcade System Boards that are using PC hardware. If you're curious on what specifications that one of the arcade system boards is used or if you're trying to match or exceed the specifications from your own, click on one of the systems below.

Namco System N2

This is an arcade system board based on Debian. It uses an AMD CPU with the NVIDIA GeForce 7 Series GPU. The games used on this system are developed with NVIDIA's Cg API. This is also a competitor of Sega's Lindbergh system.

View details »

Namco System ES1

This is an arcade system board based on Arcade Linux Live, a Debian 4.0 based Linux OS. It uses an Intel Core 2 Duo CPU with the NVIDIA GeForce 9600 GT. The games used on this system are developed under the NVIDIA Cg Shading API. There is an A2 revision of the ES1 used for overseas arcade content.

View details »

Namco System ES3

This is an arcade system board that is running Windows Embedded 7 Standard in either 32-bit or 64-bit depending on the game. It uses an Intel Core CPU with the NVIDIA GeForce GTX 600 series. The games used on this system are developed with Direct3D 11 API. There are three revisions: A, B & X.
